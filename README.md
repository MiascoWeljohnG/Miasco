# 🎮 Dodge the Creeps

## 🧾 Description
**Dodge the Creeps** is a fast-paced survival dodge game developed using the Godot Engine.  
Players take control of a lone survivor — a mysterious man wearing a traditional kimono — trapped in a haunted arena filled with relentless supernatural threats.

Your goal is simple: **survive for as long as possible.**

Face endless waves of eerie, fast-moving enemies:
- 👻 Ghosts that glide unpredictably across the screen  
- Each spirit moves erratically, forcing quick reflexes and constant awareness  

The game challenges players to master movement, timing, and focus under pressure.

---

## ✨ Features
- ⚡ **Dynamic Dodge Gameplay** – Survive against continuous waves of ghostly enemies  
- 👻 **Haunting Enemy Design** – Unpredictable floating ghosts with erratic patterns  
- 🎯 **Responsive Movement System** – Smooth 4-directional movement with animation blending  
- ♾️ **Endless Survival Mode** – The longer you survive, the higher your score  
- 🏆 **Real-Time Score Tracking** – Beat your personal best  
- 🧠 **Simple but Challenging Mechanics** – Easy to learn, hard to master

## 🛠️ Development Notes
This project was built as a learning experience focused on core game development systems, including:

- Real-time player movement  
- Collision detection systems  
- Enemy spawning logic  
- Animation control  

### ⚙️ Challenges Faced
One major issue encountered during development:

- **Collision detection inconsistencies after restarting the game**

### 🔧 Solutions Implemented
- Adjusted physics handling and node reset logic  
- Created a custom **4-direction movement checker system**  
  - Improved animation transitions  
  - Ensured smoother and more consistent controls
